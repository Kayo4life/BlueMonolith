# BlueMonolith

Hi there! Blue Monolith is a SwayWM rice, aiming for the vibe of retrofuturism.

Imagine yourself the yet to be very important scientist working at IBM. You hit breakthrough, and the first personal computer that you were working on for months finally posts. It's... It's beautiful. You imagine to yourself what the computers of the future will be like, and, this rice is what you picture.
